# 🎓 Student Performance Prediction System


📌 Dataset Link:
https://www.kaggle.com/code/guna19012000/starter-students-performance-in-exams-f48da9f2-2

## 📌 Project Overview
This project is a Machine Learning-based web application that predicts a student's math score based on academic and personal factors such as reading score, writing score, gender, lunch type, and test preparation.

The model is deployed using Streamlit to provide an interactive user interface.

---

## 🚀 Features
- Predict student math performance
- Simple and interactive UI using Streamlit
- Machine Learning regression model
- Data visualization support
- Real-time prediction system

---

## 🧠 Machine Learning Model
- Algorithm: Linear Regression
- Task Type: Regression
- Input Features:
  - Gender
  - Reading Score
  - Writing Score
  - Lunch Type
  - Test Preparation Course
- Output:
  - Predicted Math Score
